# Terraform Modules

## What is a Module?

A module is a reusable Terraform configuration block.

Instead of repeating code: You define it once and reuse it.

------------------------------------------------------------------------

## Why Modules Are Important

-   Reusability
-   Clean structure
-   Better organization
-   Used for dev, staging, prod

------------------------------------------------------------------------

## Example Structure

infra/ modules/ bigquery/ pubsub/ iam/

Each module handles one resource group.

------------------------------------------------------------------------

## Benefits

-   Reduces duplication
-   Easier maintenance
-   Scalable architecture

------------------------------------------------------------------------

## Interview Questions

Q: Why use modules in Terraform?\
A: To make infrastructure reusable and maintainable.
